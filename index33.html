1. What are the different ways to select an element in the DOM?

The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

2. What is the difference between innerHTML, innerText, and textContent?

textContent returns all text contained by an element and its children for formatting purposes. innerText returns all visible text inside an element. innerHTML returns text along with any HTML tags inside the element.

3. What is event delegation in the DOM?

Event delegation is a JavaScript technique where a single event listener is attached to a parent element to manage events for multiple child elements.

4. What is event bubbling in the DOM?

Event bubbling occurs when an event triggered on a child element propagates up through the DOM, activating event handlers on parent elements.
